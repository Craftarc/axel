axel::Handler::Handler ( const std::string &  environment)

Parameterised constructor. Sets up the Handler with the correct configurations, depending on if the code is meant to be run in the production environment or local testing.

Sets up the handler with the correct environment variables and configurations. For example, selecting "local" would mean that database queries are made to testing databases instead of live ones.

Parameters
environmentThe environment this handler runs in. Can be either "production" or "local"
