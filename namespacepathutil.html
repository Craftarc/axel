Adds query parameters to a url string.

Adds query parameters to given URL.

Works with url strings regardless of if they have existing query parameters.

Parameters

parametersA map of keys and values to be added to the URL.
Returns
A URL string with the specified parameters appended.
Parameters
urlTarget URL reference, will be modified by adding query parameters.
parametersMap containing pairs representing query parameters and their values.
Returns
Updated URL with added query parameters. Returns original URL if input parameter map is empty.
